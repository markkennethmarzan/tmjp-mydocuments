What is CCP?

Amazon Connect is a cloud-based contact center solution. It provides rich metrics and real-time reporting that allow you to optimize contact routing.

Logging in to Amazon Connect (as an admin)

  • To start using Amazon Connect, the admin should first log into their account. Following this link will redirect you to the log in form of your account:

    User Acceptance Testing (UAT) Link: https://tmjccs-test.awsapps.com/connect/home

    Figure 1. Amazon Connect Log-In Form

  • Type your username and password in the input boxes. Then, click on the blue “Sign In” button.
